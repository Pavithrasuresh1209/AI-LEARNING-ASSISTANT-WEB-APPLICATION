 AI-Powered Learning Assistant
A modern, full-stack web application designed to optimize the study experience. This platform transforms static study materials into interactive learning assets, including summaries, flashcards, and quizzes using advanced AI integration.

🚀 Features
AI Document Summarization: Instantly condense long PDF or text materials into key takeaways.
Interactive Flashcards: Automatically generate study cards from uploaded content.
Smart Quiz Generation: Test your knowledge with AI-generated questions based on your specific study materials.
Progress Tracking: Monitor your learning journey through an intuitive dashboard.
Resource Management: Organize and store your study materials efficiently in the cloud.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS, Lucide React (Icons)
Backend: Node.js, Express.js
AI Integration: Google Gemini API
Database: MongoDB (User profiles and study data)
